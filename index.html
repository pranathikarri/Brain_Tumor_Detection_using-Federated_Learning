⚠️ Note : Only MRI images in JPG or JPEG format are supported

About This Project :

Our Brain Tumor Detection System uses advanced deep learning models to scan MRI images and detect tumors with high precision. This project leverages Federated Learning to ensure patient data privacy while still enabling collaborative model training. The web app is intuitive, lightweight, and requires no medical expertise to operate. It empowers healthcare professionals and researchers to rapidly assess MRI scans from anywhere.

Key Advantages :

  • 🔒 Privacy-Preserving: Uses federated learning to protect patient data.
  • ⚡ Instant MRI Scan Predictions: Classifies brain MRI images in under 2 seconds.
  • 🎛️ Simple, Intuitive UI: No technical skills needed to use the tool.
  • 📈 High Accuracy & Reliability: Trained with extensive brain tumor datasets.
  • 🌐 Web-Based & Portable: Use it from anywhere without installing complex software.
